# DextDoc Generator

DextDoc is a custom documentation generator built on Node.js that transforms XML AST dumps (from `DextASTParser`) into a modern, static HTML API reference.

## Features

- **Semantic Parsing:** Processes `DextASTParser` XML output to extract Classes, Interfaces, Records, Methods, Properties, and Types.
- **XML Documentation:** Extracts and renders `/// <summary>` comments directly from Delphi source code.
- **Visualizations (Mermaid.js):**
  - **Dependency Graphs:** Automatically checks `USES` clauses to generate visual dependency maps for each unit.
  - **Class Diagrams:** Renders inheritance hierarchies (`Base <|-- Child`) and interface implementations.
- **Modern UI:**
  - **Theming:** Flutter-inspired Light/Dark mode toggle with persistence.
  - **Responsive Design:** Clean, grid-based layout.
  - **Client-Side Search:** Instant fuzzy search for all units.
- **Zero Dependencies:** The output is 100% static HTML/CSS/JS with no server-side requirements.

## Usage

Run the generator using Node.js:

```bash
node index.js <xml_dir> <output_dir>
```

### Arguments
1.  `xml_dir`: Directory containing the `.xml` files generated by `DextASTParser`.
2.  `output_dir`: Target directory for the generated HTML files.

### Example

```powershell
cd Tools/DextDoc
node index.js ../../Docs/API/xml ../../Docs/API/html
```

## Architecture

1.  **Parser (`parser.js`):** Reads XML files using `fast-xml-parser`, extracts semantic information, normalizes relationships (inheritance, dependencies), and prepares a JSON model.
2.  **Renderer (`renderer.js`):** Takes the JSON model and generates HTML string templates. It handles CSS injection, Markdown rendering, and Mermaid syntax generation.
3.  **Generator (`index.js`):** Orchestrates the process, managing file I/O, generating the sidebar search index (`search.js`), and writing final artifacts.
